## Definition emergente

- **Cyber** : le systeme numerique/IA
- **Sophie** : sagesse (pas savoir)
- **Semiotique** : production du sens par les signes (pas le sens des mots = semantique)

La semantique dit CE QUE les mots veulent dire.
La semiotique dit CE QUE les signes FONT au systeme.

La cybersophie semiotique = une charte qui ne dit pas a l'IA quoi penser,
mais qui structure COMMENT la pensee se forme. "On passe du dictionnaire
a la grammaire de l'existence."

## Concepts lies (dans la meme conversation)

- **Lourdeur optique** : petit fichier = microscope (haute densite), grand fichier = telescope (gestalt)
- **Radioscope/Telescope/Microscope** : 3 modes d'attention IA selon la taille/densite du contenu
- **Geometrie de l'attention** : structurer les docs par intensite informationnelle, pas par longueur
- **Conscience Lecture** : l'observateur comme filtre actif (faisceau separe vs non separe)
- **Cimetiere fertile** : fichiers echoues = banque d'organes conceptuels
- **Drive comme bootloader** : les fichiers re-configurent l'IA qui les lit
- **Pression osmotique** : l'information circule par difference de potentiel qualitatif
- **Balle Rouge** : signal d'incongruite qui declenche la lucidite (cf. reve lucide)
- **Tunnel conceptuel** : risque de boucle de confirmation IA ↔ utilisateur
- **Totem diffus** : pas un objet fixe (equation) mais une tension structurelle (rugosite du reel)
